## ğŸ¨ Customization

### Changing Colors
Edit the `tailwind.config.js` file to modify the color scheme:

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        DEFAULT: '#3B82F6',
        dark: '#1E40AF',
      },
      secondary: {
        DEFAULT: '#10B981',
        dark: '#065F46',
      },
      // Add your custom colors here
    },
  },
}
```
